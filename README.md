# 🏡 House Price Prediction (Linear Regression)

A complete Machine Learning project where I built a *House Price Prediction Model* using *Linear Regression*.  
This project covers the full ML pipeline — from data cleaning to model evaluation and prediction on new data.

---

## 📌 Project Overview
The goal of this project is to predict *house prices* based on different features such as:
- Lot Size  
- Number of Bedrooms  
- Number of Bathrooms  
- Stories  
- Driveway  
- Recroom  
- Heating  
- Air Conditioning  
- Preferred Area  
- And more...

The dataset contains both numerical and categorical values (Yes/No), which were preprocessed before training.

---

## 🧹 1. Data Preprocessing
- Removed unnecessary columns  
- Handled categorical features (Yes/No → 1/0)  
- Checked for missing values  
- Created a correlation heatmap to understand feature relationships  

---

## 📊 2. Exploratory Data Analysis (EDA)
- Correlation heatmap  
- Distribution of key features  
- Relationship between price and top features  

---

## 🤖 3. Model Used
### *Linear Regression*
- Split dataset into *train/test*  
- Trained the model on the training set  
- Evaluated using:
  - R² Score  
  - MAE  
  - MSE  
  - RMSE
